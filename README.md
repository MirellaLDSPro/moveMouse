# moveMouse
Robot to move mouse randomly across the screen.

## Setup
 - Install Python
 - Add the dependencies:
    ### Windows
      - pip install PyAutoGUI
    ### Mac
      - pip3 install pyobjc-core
      - pip3 install pyobjc
      - pip3 install pyautogui
     ### Linux
      - pip3 install python3-xlib
      - pip3 install pyautogui
